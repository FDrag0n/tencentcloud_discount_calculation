# tencentcloud_discount_calculation
腾讯云优惠券计算工具

由于腾讯云发的优惠券都是满减金额券，数量大了难免不知道折扣力度，作为穷鬼肯定选力度最大的，所以用hunyuan帮忙糊了一个脚本，使用油猴等工具加载即可运行，监听了优惠券表格变动事件

顺路托管到了greasyfork，可以点击这里安装
